About

QuoteSkye is a multimedia application to dynamically generate random quotes, including a random image with the overlaid text.

The application:

  • Reads quotes from a variety of filetypes (PDF, Word Documents, CSVs, Text files).
  • Loads, manipulates, and saves images.
  • Accepts dynamic user input through this Flask web service and a command-line tool.

Images are from the Isle of Skye in Scottland. Source: unsplash.com

Built With

Python, Flask, xpdf, Bootstrap, Git
